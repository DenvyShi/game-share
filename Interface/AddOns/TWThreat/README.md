光环
在默认 UI 的单位框架上显示从绿色/黄色/红色变化的彩色光环。

光环的颜色基于您当前的威胁阈值：

绿色 - 黄色：0% - 49%
黄色 - 红色：50% - 100%
光环


全屏光环
当您的威胁超过 80% 时，屏幕边缘闪烁红光。

此功能在 Tank Mode 中禁用。



仇恨警告声
当您达到 99% 的威胁时会播放警告声音。

此功能在 Tank Mode 中禁用。
