# Rares
Rare npcs scanner and tracker - Ace2 version<br/>

*(inspired by [npcscan](https://github.com/shirsig/npcscan) by shirsig and [SilverDragon](https://github.com/cdmichaelb/SilverDragon) by cdmichaelb)*<br/>

This addon automatically scans for rares and alert you with two sound effects and two icons: one when found alive and one when found dead.

Current release: 0.6.3
What's new?<br/>
- Fubar tooltip fix.

### NOTE: Saved variables from version prior to 0.3.0 will be lost

Slash commands (also through FuBar menu):
- **/rares** : show all available commands
- **/rares buttonLock** : Lock/Unlock Rares radar frame
- **/rares buttonShow** : Show/Hide Rares radar frame
- **/rares tooltipAnchor** : Toggle Rares radar frame tooltip anchor LEFT or RIGHT
- **/rares showModel** : Show/Hide Model frame
