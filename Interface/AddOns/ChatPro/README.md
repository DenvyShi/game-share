# ChatPro

ChatPro 是一款专为魔兽世界 1.12.1（乌龟服/Turtle WoW）设计的聊天窗口全面增强插件。它旨在提供更现代、更便捷的聊天体验，重构自经典的 ChatMOD 插件。

## 主要功能

- **职业着色与等级显示**：在聊天窗口中自动为玩家姓名标注职业颜色，并显示其等级（支持按等级差距着色）。
- **自动查询 (Auto Who)**：遇到未记录的玩家发言时，插件会自动后台发送 `/who` 查询并缓存其职业和等级信息。
- **正文高亮**：如果聊天内容中出现了已知的玩家姓名，插件会自动将其着色。
- **频道名简化**：将繁琐的频道名称简化（例如：`[公会]` 变为 `[会]`，`[寻找组队]` 变为 `[4]`），并支持自定义频道映射。
- **鼠标滚轮增强**：
  - 直接滚动鼠标滚轮即可翻页。
  - **Shift + 滚轮**：快速翻页（一次跳过多行）。
  - **Ctrl + 滚轮**：直接跳转至页首或页尾。
- **时间戳功能**：为每条消息添加时间戳，点击时间戳可弹出复制框。
- **网址识别**：自动识别聊天中的 URL 链接，点击链接可弹出复制框以便在浏览器中使用。
- **输入框增强**：
  - 支持将输入框置顶。
  - 增强历史记录（支持多达 250 条记录）。
  - 无障碍编辑：无需按住 Alt 即可使用方向键移动光标。
- **粘性频道**：自动记忆上次使用的频道（说、小队、团队、公会、密语等）。
- **错误屏蔽**：自动隐藏屏幕中央常见的红字提示（如“技能还没有准备好”、“超出射程”等）。
- **界面优化**：可选择隐藏聊天窗口旁边的翻页按钮和菜单按钮，使界面更清爽；支持禁用聊天文字渐隐。

## 使用说明

### 常用命令
*   `/chatpro`：打开或关闭插件设置界面。

### 设置界面
在设置界面中，你可以根据个人喜好开启或关闭上述功能，并进入“频道简称设置”来自定义各个频道的缩写。

## 安装方法
1. 下载并解压。
2. 将 `ChatPro` 文件夹放入魔兽世界客户端目录下的 `Interface\AddOns` 文件夹内。
3. 确保文件夹路径为 `Interface\AddOns\ChatPro\ChatPro.toc`。

## 注意事项
*   本插件主要针对 1.12.1 版本开发，完美适配乌龟服（Turtle WoW）中文环境。
*   如果使用其他聊天增强插件，可能会存在冲突，建议关闭同类插件后再使用。
